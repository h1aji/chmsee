About

Since the developement discontinued by main developer I have decided to continue maintenance and further developement on Github.
The original project were hosted on Google Code

ChmSee is an HTML Help viewer for Unix/Linux.

ChmSee have got all the required functionality for regular use. Bookmarks, search, print and treeview are availbale.


News

2016-01-25. 1.4.0 Release. Webkit port based on GTK2 libraries. Next release will include GTK3 migration.
